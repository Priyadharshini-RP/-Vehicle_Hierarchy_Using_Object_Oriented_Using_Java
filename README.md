# -Vehicle_Hierarchy_Using_Object_Oriented_Using_Java

🚗 **Java Vehicle Management Program**

I created a simple Java program to demonstrate Object-Oriented Programming concepts using vehicles like cars and bikes.

🔹 Features:
• Create a **Vehicle class** with brand and seat details
• Implement **inheritance** with Car and Bike classes
• Use **getters and setters** for accessing and updating data
• Display vehicle details using class methods

💡 OOP Concepts Used:
• **Inheritance** – Car and Bike inherit from Vehicle
• **Encapsulation** – Getter and setter methods
• **Constructors** – Used to initialize vehicle objects
• **Method Reuse** – Display method used by child classes

This project helped me understand how inheritance and encapsulation work in Java.
